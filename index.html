Seminário 1: Displays e Positions

Empresa.inc

O display block posiciona os itens em uma linha ocupando todo o espaço horizontal possível

O display flex serve para manter os itens alinhados na vertical

O display inline serve para criar apenas o espaço necessário para o conteúdo

O display grid serve para fazer os elementos se comportarem como grade com linhas e colunas

O display table serve para fazer os elementos se comportarem como tabelas composta por linhas e colunas

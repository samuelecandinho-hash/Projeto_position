O display inline block permite que itens fiquem em linha mas também respeitem suas respectivas proporções

O display inline flex alinha os itens de forma horizontal ao mesmo tempo que a div pai só ocupa o espaço necessário

O display inline-grid cria uma grade semelhante ao display grid, porém a div pai tem a propriedade de inline que faz com que ela so tenha tamanho necessário para o conteúdo se encaixar. Com o display grid padrão, é possível que elementos fiquem fora da div pai ou que sobre espaço na mesma.

O display flow-root faz o contêiner se envolver com elementos filhos

usando o block-Flex, podemos alinhar itens na horizontal dentro desse espaço. objetos não vão para a próxima linha, e atravessam o espaço.

O block grid transforma um elemento em um contêiner de grade, onde pode organizar elementos filhos em linhas e colunas ocupando todo o espaço possível no elemento.

O block flow posiciona itens na ordem que estão no contêiner em uma linha horizontal ocupando todo o espaço possível no elemento.
